KataGo棋譜検索システム

棋譜フォルダに入っているkatago-rating-gameの中から検索します！

緑色の棋譜リストから棋譜を選択すると閲覧できます。ホイールで前後に手順を進めることができます。

KataGoエンジンの形勢判断が背景の色でわかります。（グレーは互角）

最後に着手した側の判断です。 -> 最終手マークが白石についていたら、白側のエンジンの判断となります。

基本操作

  • 碁盤の上でドラッグすると検索したい範囲を指定できます。（ドラッグ中に右クリックでキャンセル）
  • 右クリック長押しで直前に指定した範囲を取り除くことができます。
  • 着手モードでは、右クリックで石を置くことができます。
  • 続けて右クリックすることで色を変えることができます。
  • 灰色の石はどちらかの石があればヒットします。
  • 「？」の付いた石は、たとえば黒石に「？」がついていれば、【黒石があるか、もしくはその地点に石が存在しないとき】にヒットします。（つまり白石があるときだけ、マッチしない）

検索機能

「検索」ボタンで検索を開始します。

  • もう一つのリストボックスに検索結果が表示されるので、クリックして閲覧してください。
  • 抜かれた部分はそこに石をおいた状態で、検索すればOKです。
  • 棋譜リストボックスの下のスピンボックスは、棋譜を選択したときに、何手目から開始するかを設定します。
  • 検索結果の下のスピンボックスは、検索結果の棋譜を選択したときに、マッチ局面の前後何手目を表示するかを設定します。

特殊検索機能

「除外検索」ボタン

検索結果を除いた全ての棋譜から検索します。

たとえば、ある局面で、次の一手が打たれなかった局面を検索したい場合：

  1. 次の一手を打った局面を検索
  2. そのあと一手もどして、「除外検索」

このようにすれば、その手を打たなかった棋譜だけを抽出できます。

「絞込検索」ボタン

検索結果の中だけからさらに検索を行います。

「厳密検索」

抜き跡にさらに着手した局面も検索可能です。

遅いです！（非推奨）

※【？石】の使用には対応していません。

「反転検索」

マッチしなかった棋譜が結果に追加されるようになります。

全ての検索方式に対応しています。

※位置ジャンプなどは行えません。フィルター用途向けです。

グルーピング機能

A-Dまでのボタンは石をグループ化して、そのうち１つだけある局面をマッチさせることができます。

実用例

黒星、白カカリ、黒ハサミのとき、いろんなハサミの位置に黒石を置き、それらを全て同じ記号、例えばA、を指定すると、そのAの中から１つだけマッチした（どこかに黒がハサんだ）棋譜を検索します。

現状では【？石】に対する動作は未定義です。

現状では、厳密検索ではグルーピング対応していません（無視されます）。

SGF取得

選択中の棋譜をクリップボードにコピーします。

  • Ctrl + 左クリックで複数選択ができます。
  • Ctrl + Aで全て選択します。
  • 複数選択されている場合それらを全てマージしたSGFがクリップボードにコピーされます。

保存ボタン -> 次回起動時まで【save】フォルダ内に、sgfファイルとして保存します。ファイルとして利用したいときに使ってください。

自動再生

自動再生のチェックを入れると選んだ棋譜を自動再生します。

スライダーで一手ごとの待ち時間を設定します：

  1. １番目のスライダーで、基本待ち時間を設定します。（100ms - 10000ms）
  2. ２番目のスライダーで、各エンジンの形勢判断の差が大きいほど待ち時間を延ばします。（0-10000ms）
  3. ３番目のスライダーで、直前の手との距離が遠いほど、待ち時間を延ばします。(0-10000ms)

１手ごとの待ち時間は全てのスライダーの値を合わせた値になります。

レイアウト変更

CTRLをダブルクリックするとデバッグモードに入ります。もう一度やれば戻ります。

グリッドが表示されるのでつまんで移動できます。

また各エリアのパディング部をつまむとエリアが選択状態になり、他のエリアのパディングをクリックすると場所を交換します。

注意事項

  • 起動後少しずつ棋譜を読み込んでいるため、起動して検索した場合は多少時間がかかります。
  • 厳密検索は更にノードに碁盤を作成するのでかなり遅いです。
