katago棋譜検索システム

棋譜リストフォルダに入っているkatago-rating-gameの中から検索します！

棋譜リストから棋譜を選択すると閲覧できる。ホイールで前後に手順を進めることができます

katagoエンジンが形勢判断が背景の色でわかります。（グレー互角)
最後に着手した側の判断です。 -> 最終手マークが白石についていたら、白側のエンジンの判断。

着手チェックを外した状態で、碁盤の上でドラッグすると検索したい範囲を指定できる。（複数組み合わせて指定可能）
右クリックで直前に指定した範囲を取り除きます。

「検索」ボタンで検索を開始します

下のリストボックスに検索結果が表示されるので、クリックして閲覧してください。

★★　レイアウト変更対応

	CTRLをダブルクリック？するとデバッグモードに入ります。もう一度やれば戻ります。
	グリッドが表示されるのでつまんで移動できます。
	また各エリアのパディング部をつまむとエリアが選択状態になり、他のエリアをクリックするとスワップ（場所を交換）します。
	save　load ボタンで現在のデザインを保存、読み込みできます。
	また、save loadボタンは右ドラッグで移動できます。

--注意事項--

-起動後少しずつ棋譜を読み込んでいるため、起動してすぐに検索した場合は時間がかかることがあります。

-高速化のため、棋譜内で石が盤上から消える状態を経過した盤面は正しく検索できません。

	例えば３３入りからの二段ハネ定石では、一目を抜く直前の形で検索してください。

	厳密モードは、だいぶ検索が遅くなることが予想されます。
	もし要望があれば実装します。
